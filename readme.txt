== Description ==

Protect your WordPress blog against image theft with Plaghunter. Plaghunter is a reverse image search that checks where your images are being used online.

This plugin lets you add images to Plaghunter directly from your WordPress backend.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plaghunter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
